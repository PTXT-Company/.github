## Sobre nós

Somos uma empresa de tecnologia focada em desenvolver soluções inovadoras para o mercado digital. Atuamos em três frentes:

| Área | Descrição |
|------|-----------|
| **Web & Software** | Desenvolvimento de aplicações web modernas e sistemas escaláveis |
| **Jogos Virtuais** | Ferramentas e soluções para o universo de games, com foco em Roblox |
| **Produtos Digitais** | Criação e monetização de ativos e produtos digitais |

## Projetos

### Make-R

> Editor profissional de roupas para Roblox — em desenvolvimento.

Canvas 2D com edição precisa via Fabric.js, preview 3D em tempo real, sistema de camadas, ferramentas completas (pincel, preenchimento, texto) e exportação otimizada para upload direto no Roblox.

### PTXT Pass

> Plataforma de cursos para criação de ativos digitais e construção de renda online.
